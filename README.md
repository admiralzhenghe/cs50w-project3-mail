# Mail

Design a front-end for an email client that makes API calls to send and receive emails.


## Architecture

**Frontend**

- Languages:

  - HTML
  - CSS
  - JavaScript

- Framework/Libraries:
  - Bootstrap

**Backend**

- Language:

  - Python

- Framework/Libraries:

  - Django

- Database:
  - SQLite
